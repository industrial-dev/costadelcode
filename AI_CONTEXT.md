# Costa del Code - Contexto del Proyecto

## Objetivo

Crear una landing page sencilla, bonita y estética para impulsar una comunidad de desarrolladores en Estepona.

## Audiencia

- Desarrolladores y personas técnicas en Estepona y alrededores.
- Interesados en networking, eventos y colaboración local.

## Propuesta de valor

- Punto de encuentro local para compartir experiencias y crecer profesionalmente.
- Comunidad abierta con actividades y eventos.

## Secciones requeridas

- Hero
- About / Propósito
- Events / Actividades
- FAQ
- CTA principal
- Footer
